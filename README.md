# Cipher Price Index

## Requirements

* iOS 12.2
* XCode 10.2 (10E125)
* [Cocoapods](https://cocoapods.org/)

## Buld & run

Inside repository root directory run:

````bash
  pod install
````
Then open **`Cipher Price Index.xcworkspace`**.
